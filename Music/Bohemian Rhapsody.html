Bohemian Rhapsody

Background

I originally arranged this piece both because I really liked the song and because I wanted to test out the new Musescore 4 and some of its new features that had just come out. Originally I had tried to saved the file using the new "Save to the Cloud" feature. Unfortunately, my file had somehow corrupted, which had set me back many hours of work. In the end, I was still able to recover all my lost information and decided to save everything locally.

Because this song was never released to anyone in particular, it was never performed.
